# broll-checklists

Checklists interactivos de tomas de apoyo (b-roll) para rodajes en Social Eco Agency.

Cada HTML es self-contained: checkboxes con persistencia local, vista previa del guión, link a la referencia de TikTok y link directo a la tarea de Asana.

Generados con la skill `/broll-checklist` desde Claude Code.

## Cómo se hostea

GitHub Pages sirve los archivos directamente desde `main`. URL pattern:

```
https://juansanchezsocial.github.io/broll-checklists/{archivo}.html
```
